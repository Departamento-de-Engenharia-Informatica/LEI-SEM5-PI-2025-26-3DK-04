## 2. Analysis

### 2.1. User Story Overview

**ID:** US_2.2.10
**Title:** View Status of Submitted Vessel Visit Notifications

**Description:**
As a Shipping Agent Representative, view the status of all submitted Vessel Visit Notifications so that decisions of the Port Authority are visible. Representatives can also view notifications from other representatives within the same organization.

### 2.2. Business Rules
1. Only authenticated users with role Shipping Agent Representative can view notifications.
2. Representatives can view their own notifications and notifications from representatives in the same organization.
3. Notifications must be searchable by vessel, status, representative and time period.
4. Results must be ordered by submission date (most recent first).
5. Each notification must track which representative created it and when.

### 2.3. Edge cases
- Filter with no results — return empty list.
- Representative not part of any organization — only see own notifications.
- Invalid status filter value — reject with validation error.
